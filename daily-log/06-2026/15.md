## Office Learning

### Apache Kafka Hands-on

Worked on a Spring Boot application using Apache Kafka.

#### Topics Covered

* Apache Kafka setup
* Broker configuration
* Producer implementation
* Consumer implementation using @KafkaListener
* DTO mapping
* JSON validation
* Entity creation and persistence

#### Flow Implemented

1. Producer reads records from a file.
2. Each record is converted into a DTO.
3. DTO is published to a Kafka topic.
4. Consumer listens to the topic using @KafkaListener.
5. JSON payload is validated.
6. Entities are created from valid records.
7. Duplicate records are checked.
8. Unique records are persisted to the database.

#### Key Learnings

* Difference between producer and consumer responsibilities.
* Event-driven processing using Kafka.
* Data validation before persistence.
* Handling duplicate records before database insertion.
